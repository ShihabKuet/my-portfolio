# Shihab's Developer Portfolio

A modern, responsive developer portfolio built with Next.js 16, TypeScript, and Tailwind CSS v4.

## 🚀 Tech Stack

- **Framework:** Next.js 16 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS v4
- **Animations:** Framer Motion
- **Icons:** Lucide React + React Icons
- **Deployment:** Vercel

## 📦 Getting Started
```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## 📁 Project Structure
```
src/
├── app/          # Next.js App Router pages
├── components/   # Reusable UI components
│   ├── layout/   # Navbar, Footer
│   ├── sections/ # Page sections (Hero, Skills, etc.)
│   └── ui/       # Small reusable elements
├── data/         # All personal content (single source of truth)
├── lib/          # Utility functions
└── types/        # TypeScript interfaces
```
